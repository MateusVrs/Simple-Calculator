<h2 align='center'>About</h2>
<h3>● Calculations</h3>
<p>→ To do the power operation you need to use "^", "e+" or "e-" symbols
<p>→ You can't use a comma to write decimal numbers, only dot</p>
<p>→ The calculator can do simple signals operations inside the expression, with "+" and "-"</p>
<p>→ To write the minus sign in front of parentheses, specify the -1</p>
<p>Ex: -1*(5+5)</p>
<p>→ To write the expressions don't use spaces between the elements</p>
